# User Guide - AI-Powered Study Tasks

Learn how to use EduFlow AI's automatic task generation and timer integration features.

---

## üéØ What's New

### Automatic AI Task Generation
When you create an event, EduFlow AI automatically:
- Analyzes your event details (title, subject, prep time)
- Generates 4-7 optimized study tasks
- Assigns duration, difficulty, and tips to each task
- Schedules tasks across available days

### One-Click Timer Integration
Each study task can now:
- Start the focus timer with one click
- Automatically track your study time
- Mark tasks as complete when timer ends
- Update your progress in real-time

---

## üìù How to Use

### Step 1: Create an Event

1. **Navigate to Events**
   - Click "Events" in the top navigation
   - Click "Create New Event" button

2. **Fill in Event Details**
   ```
   Title: Python Midterm Exam
   Type: Exam
   Date: [7 days from now]
   Subject: Computer Science
   Priority: High
   Prep Time: 6 hours
   Description: Focus on OOP, data structures, and algorithms
   ```

3. **Click "Create Event"**

**Result:** AI automatically generates study tasks!

---

### Step 2: View Your AI-Generated Tasks

After creating the event, you'll see:

```
Task 1/4: Review key concepts and fundamentals

Type: Concept Review
Difficulty: Low
Duration: 30 minutes

Tips:
Start with understanding core principles
```

Each task includes:
- ‚úÖ **Task number** - Shows progress (1/4, 2/4, etc.)
- ‚úÖ **Task title** - What to study
- ‚úÖ **Type** - Concept review, practice, deep practice, revision, or mock test
- ‚úÖ **Difficulty** - Low, medium, or high
- ‚úÖ **Duration** - Optimized time (25-60 minutes)
- ‚úÖ **Tips** - AI-generated study guidance

---

### Step 3: Start Studying with the Timer

1. **Find a pending task** on the event detail page

2. **Click "Start with Timer"** button

3. **What happens automatically:**
   - Focus timer starts
   - Task status changes to "in_progress"
   - Timer displays event name and duration
   - Break recommendations enabled

4. **Study according to the task description**

5. **When done:**
   - Click "End Session" in focus timer
   - Task automatically marked "completed"
   - Actual study time recorded
   - Break recommendation shown

## üé® Understanding Task Types

### Concept Review
**Purpose:** Learn new material
**Activities:** Reading, watching lectures, understanding fundamentals
**Difficulty:** Usually low to medium
**Duration:** 25-30 minutes

### Practice
**Purpose:** Apply what you learned
**Activities:** Solving problems, exercises, applying concepts
**Difficulty:** Medium
**Duration:** 30-45 minutes

### Deep Practice
**Purpose:** Master complex scenarios
**Activities:** Challenging problems, projects, critical thinking
**Difficulty:** High
**Duration:** 45-60 minutes

### Revision
**Purpose:** Review and consolidate
**Activities:** Re-reading notes, reviewing formulas, filling gaps
**Difficulty:** Low
**Duration:** 25-30 minutes

### Mock Test
**Purpose:** Simulate exam conditions
**Activities:** Practice tests, timed exercises
**Difficulty:** High
**Duration:** 45-60 minutes

## üí° Pro Tips

### Getting the Best AI Tasks

1. **Be specific in descriptions**
   ```
   Good: "Focus on OOP, data structures, and algorithms"
   Basic: "Study for exam"
   ```

2. **Set realistic prep time**
   - Too little: Tasks rushed
   - Too much: Tasks spread too thin
   - Just right: 2-3 sessions per day max

3. **Use appropriate event types**
   - Exam ‚Üí Includes mock test
   - Quiz ‚Üí Quick review and practice
   - Assignment ‚Üí Deep practice focused

### Managing Your Study Sessions

1. **Follow the difficulty progression**
   - Start with low difficulty (warm up)
   - Build to high difficulty (challenge)
   - End with low difficulty (consolidate)

2. **Take recommended breaks**
   - After 1 session: 5-minute break
   - After 2 sessions: 10-minute break
   - After 4 sessions: 15-20 minute break

3. **Mark skipped sessions honestly**
   - Don't skip to show false progress
   - System tracks for better recommendations

## ‚ùì FAQ

### Q: Can I create events without AI tasks?
**A:** Yes! If AI is disabled, the system uses smart deterministic scheduling.

### Q: Can I edit AI-generated tasks?
**A:** Yes! Click "Details" on any task to edit the content or duration.

### Q: What if I don't finish a task in the suggested time?
**A:** No problem! The actual time is tracked. If you consistently overrun, AI can suggest adjustments.

### Q: Can I start a task without the timer?
**A:** Yes! There's a separate "Start" button for basic tracking without the timer.

### Q: Do I need to complete tasks in order?
**A:** No! But following the AI's difficulty progression is recommended for best results.

### Q: What if I miss a task?
**A:** Mark it as "Skipped". The system will adapt and may redistribute remaining tasks.
